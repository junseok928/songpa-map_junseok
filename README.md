# 송파구 토지이용 분석

가천대학교 스마트시티학과 국토 데이터분석 실습 (Lecture 6)  
학번: 202234678 / 이름: 양준석

---

## 개요

서울특별시 송파구를 대상으로 필지 단위 건축물 주용도, 용도지역, 행정동별 인구·가구 현황을 지도 위에 시각화한 웹 시스템입니다.

공간 범위는 송파구 전체이며, 건축물대장(2026.06), 연속지적도(2026.05), SGIS 집계구 인구통계(2024년 기준) 데이터를 활용했습니다.

---

## 사용 데이터

- 건축물대장 — 건축HUB (https://www.hub.go.kr)
- 연속지적도형정보 — 국토교통부 / Vworld
- 집계구 인구·가구 통계 — SGIS 포털 (https://sgis.mods.go.kr)
- 베이스맵 — Vworld WMTS

---

## 구현 내용

지도에서 필지를 클릭하면 주용도, 용도지역, 대지면적, 연면적, 층수, 사용승인일을 확인할 수 있습니다.

레이어는 건축물 주용도 / 용도지역 두 가지로 전환 가능하며, 우측 패널에서 용도별 통계표와 행정동별 인구 현황을 볼 수 있습니다.
